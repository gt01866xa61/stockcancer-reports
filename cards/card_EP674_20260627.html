市場動態
週五全面恐慌、勇氣盡失——連孟恭都認加碼太快
🎯 很多熱門族群跌破季線,做多意願轉弱;孟恭手上強勢漲價元件離季線遠,但若週一續跌會反手認賠。
🔍 本以為週三、週四就反彈:週四 11 點半後台股中小與強勢股拉尾盤,孟恭已把錢打進去。週五早盤先殺,9–10 點國巨期貨直接跌停(現股還沒跌)+ 台積電急殺——期貨先跌停代表槓桿超大的要自斷/斷頭、恐慌大逃殺。他以為籌碼洗差不多了再打部位,中午相對強勢股試著站回、他被套在日內高點,事後覺得打太快。

週五一大根讓很多過去熱門族群直接跌到季線下面:這些跌破季線的東西做多意願偏低、反彈摸到大均線會遇套牢盤解套賣壓,後面上去力道小——「做壞掉了就被丟壞掉了」。
本集持倉狀態 ⭐ 被動 + 功率元件(強勢、離季線遠、相對抗跌) ⭐ 週五加碼部位(被套日內高點)
⚠️ 孟恭原話:若下週一又開低續走,「我那個加碼的應該有把它丟出來,反正你們不要玩,老子也不要玩,協助把這個盤砸掉」——他自己就可能是賣壓。
📌 檢查自己手上部位:若一路灌破季線、反彈站不回去,即便是 AI 大題材也要小心(資金沒在這、上去難);跌破季線是價值投資相對好買點,但要等。留意現金水位、有東西要買先騰錢。

🔍 SpaceX:挪錢加碼,把表現差的持股與 Tesla 賣一賣換進——買 Tesla 原本就是為了「壓住馬斯克」,現在有 SpaceX 可以壓,角色替換。

CPU / Marvell:有壓到整個 CPU、ASIC 壓 Marvell,但近期都做了一點調節,因美股走勢分化。操作邏輯:台股轉弱就賣、美股轉弱反而接刀(砍相對強勢、換掉下來弱到不錯價位的科技巨頭)。

軟體:本以為 Palantir 這種估值超高、大家最不信想放空的會持續領頭,但近期疲弱;Portfolio 裡走得好的是 CrowdStrike——好的持續點、沒那麼好的不減碼但也不加碼。
加碼 ⭐ SpaceX SPCX CrowdStrike CRWD
減碼 / 調節 ⭐ Tesla TSLA Marvell MRVL CPU 族群
持有不動 ⭐ Palantir PLTR(疲弱、不減不加)
⚠️ Tesla 是資金調度與角色替換,非看壞基本面;Palantir 疲弱是「不加碼」而非看空。別把孟恭的調節讀成方向性看法。
📌 美股劇本:相對難做,他相信後面要轉進應用層面,巨頭有機會;最喜歡 Google(降本增效/自家晶片/配置佳)、蘋果(Edge 端領導、等轉強切入);週五跌深的微軟、蘋果若持續上去他會把錢移過去。

修正診斷
不是戰爭也不是升息,最接近的是主權基金 rebalance
🎯 排除戰爭、升息、AI 泡沫;歸因季度再平衡。另提出「巨頭股東會不會坐不住」待追蹤論述。
🔍 不是戰爭:直接看油價就好——已跌到七八十塊、是善後階段,革命衛隊新聞多偏假消息。不是升息:川普本人喜歡通膨、不會大幅升息;漲價狂潮集中在 Data center(一台 AI server 記憶體 BOM cost 已爬到 30%,但巨頭照買),消費端則是遞延消費。AI 泡沫論:聽聽就好,泡沫論者 2023 講到現在總會有對的一天、但錯過這麼大段。

最接近 = rebalance:過去一季科技股賺太多→漲多的標的淨值佔比變高→保守策略機械式減碼、挪去買淨值佔比低的,資金弱流強。
⭐ 值得追蹤的論述:巨頭股東會不會坐不住
🔍 美光毛利率 80 幾、接下來幾個 Q 甚至可能上 90 幾(暴利、隨便賣隨便賺);但這些付錢的科技巨頭股價表現不好、有些 cash flow 要跑負值,會不會出現檢討聲音「錢都付給零組件廠賺走、成效不一定看到、還要不要推這麼快」(DeepSeek 那波有類似討論但沒燒大)。
但孟恭掃了巨頭 deploy AI 伺服器數字 + 台系供應鏈拉貨,目前沒太多問題→比較像市場信心短期丟失(類 2026/3、6 月初)。
關聯 Micron MU 科技巨頭整體 Data center / AI server
📌 盯盤面是否續轉差——若轉差,「巨頭坐不住」論述可能被醞釀出來、影響 sentiment;同時盯巨頭 deploy 伺服器數字與台系拉貨有無轉弱(目前沒問題)。

消費電子
蘋果跟進漲價 10–20%,Markup 不是專利,最後看銷量
🎯 蘋果在記憶體高漲下上修 MacBook 等 10–20%;漲價剛出一度跌隔天又漲,孟恭判 neutral、要看銷量。
🔍 選在這時間漲「舊品」,可能是替新品漲價先預熱。關鍵:嚇到一些人不代表嚇到果粉——討論話題要先定義「客人是誰」、對方是不是跟你同類型(井水不犯河水,蘋果與安卓用戶彼此不影響)。

Markup 不是蘋果專利(愛馬仕加幾個鉚釘能多賣上萬、華強北包就不行):蘋果是價格黏性高、毛利肥,不用這麼快反映成本;中系 OEM 才老實反映(用低價搶市佔)。所以之前蘋果反而相對划算、衝了一波銷量;現在它跟進漲價,要看後續銷量判斷消費者吃不吃得下。
關聯 Apple AAPL Micron MU 中系手機 OEM
⚠️ 供應鏈漲價≈好消息(巨頭照買),但消費端漲價要小心——面對普羅百姓,貴 20–30% 有些就不買,血淋淋反映在銷量。
📌 判斷漲價會不會傷消費,直接看銷量這個結果(像看戰爭就看油價);蘋果漲價現在 neutral,等銷量數字出來再下判斷。

Q&A 觀念輪廓
修正時的四個元規則:鬼故事別當真、消息跟在價格後
① 鬼故事知道就好、別太認真
🔍 大盤往下 5–8% 機會很多,每次下跌都有鬼故事(戰爭/升息/泡沫),但新高後回頭看根本沒人在討論——不好的時候被拿出來當談資而已。
② 下跌蔓延時消息是跟在後面的
🔍 價格變化常跑在基本面前面,不能純 fundamental。參考去年關稅地震:3 月美股整天跌、沒消息,他的做法是「找個還 OK 的反彈、有尊嚴地出去」,後面才看到關稅地震的消息。
③ 財富效應擴張時大家其實不花錢
🔍 盤好時不把錢丟進市場「機會成本太高」,所以大家持續往市場丟、不消費;要等盤面震盪、修理人頻率變高、產生「賠掉不如花掉」的想法,消費才會發生——那時候看漲價會不會衝擊消費才準。
④ 討論話題先定義「你的客人是誰」
🔍 先定義客人是誰、對方是不是跟你同類型;看漲價先看是不是客戶已拒絕吃——供應鏈漲價=好消息,消費端漲價最後直接看銷量這個結果。

Q&A 觀念輪廓
越來越沉默、喊停的人停不下來、事業要單純
錢 vs 自由(MTK 換工作)
🔍 年輕人賺錢比較重要;工作很難非常快樂,只要不會壓力大到憂鬱,就是可接受的 trade off。覺得自己沒錢常是比較對象是資產階級(長年累積);換地方說不定一樣不滿、錢又少、不爽指數更高。
省錢心態與資產無關(York)
🔍 股票漲時沒人想把錢拿出來花(每筆都能賺更多、反感踩煞車);這心態百萬到十億都一樣不夠。只見過一個人真賺到目標後退休買債券完全退出,其餘喊停的都停不下來——要刻意找平衡、善待家人(你一天的震盪可能遠超家人許願的東西)。
越來越沉默(Pack)
🔍 不需要跟任何人講太多或說服任何人——不同位階/資產/家庭背景沒辦法對話,真正需要說服的只有自己家庭最核心的人。
事業要單純(Andy)
🔍 股東不要太複雜(複雜就麻煩);創辦人要 skin in the game(自己投錢才會百分百認真);別為拿一堆人的錢迎合一堆人、失去初衷。林口投資是社會服務學習態度(投報率沒股票好)。
