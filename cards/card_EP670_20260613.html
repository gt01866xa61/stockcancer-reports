SpaceX 上市操作 + 估值的重點是「估在別人前面」

主持人有去申購(深購價)+ 盤中 160 幾也加碼,接下來幾天會持續點一點 — 當作支持太空夢想。

類比持有特斯拉:相挺它 + 題材很酷的背景 + 目標也是賺錢,但不針對估值做太多討論。
拆解 SpaceX:衛星(賺錢引擎)+ 火箭(夢想較遠)+ AI(本夢比,若能在低軌衛星放 Data Center 或重新拋算力出租)。除衛星外都是本夢比,這種公司不用認真算估值。

估值是藝術:估在別人前面

估值的重點不是估的當下正一個對錯,是估在別人的前面 — 你買進後,後面還有源源不絕的人進來買、你可以出貨給他們、賣在更高的地方,這才是好估值。
不要比絕對數字 — 市場一堆超便宜、高殖利率但連漲都不會漲的東西。
成長股 vs 價值股界線越來越模糊:成長股的價值常放在後面,等你發現它是價值股時,已過完最甜的階段。記憶體七張點時你覺得貴、現在看很便宜;春田現在看貴,但若 EPS 狂拉漲價成功,回頭看 PE 可能 20 倍以下。
會信 SpaceX 的就會信、不信的就不信(買紡織股 vs 買太空股是兩種人),沒必要也沒辦法說服對方。Portfolio 裡有些是便宜的、有些是賣夢題材對的,看市場表現用加減碼調整。

被動元件漲價速度超預期(兩次到 terminal),料號從 AI 核心擴到周邊。業內沒大擴產→週期較長。配貨機制 run 下去會驗證需求真假,持續追蹤漲價兌現。

Anthropic Fable 吃監管 + Edge computing 那天不遠了

Fable 受美國出口管制

Anthropic Fable 模型好用、全面、速度快但 token 燒很快;但海外受美國出口管制,本來在 Perplexity 還能間接用,錄節目前測試發現也被鎖了。
「你一直講自己東西很危險很恐怖,你就是會去吃監管鐵拳。」認同 Palantir 的 Alex Karp 講的 — OpenAI 的 Sam Altman 跟 Anthropic 的 Dario 不知道自己有多討人厭(訴諸恐懼)。後果已現(海外被禁),嚴重會影響市場觸及度、某次修正會被當故事。

Edge computing 那天不遠了

前沿模型拉很強 + 開放模型落後閉鎖約半年;等閉鎖摸到新結界、開放也拉到相對強,地端算力就能完成多數人一兩年前 AI 能辦到的事。
一年回推:2028 回看 2027 模型,對絕大多數人非常夠用、可完全 run 在地端 → 地端裝置迎來爆發。
監管可能催生 Edge 趨勢:若前沿模型開始吃監管鐵拳,市場偏好可能從追大模型/大 cluster 轉去 Edge — 成長性沒這麼好但也不錯、同時籌碼乾淨、相對低基期的東西。現在摳這個可能太早,但已聞到味道。
Dario 訴諸恐懼吃監管鐵拳,後果已現。Edge computing 的那天不遠(開放落後閉鎖半年、都拉強後地端夠用)。留意若大模型吃監管,資金可能轉去籌碼乾淨、低基期的 Edge 概念。
Q&A 投資方法
7/10

資訊邊際效益遞減(投片數字是 noise)+ 洗手場追回

小菜雞收集資訊 + 投片數字是 noise

管道:多開戶拿券商報告 → 交易量大開法人戶拿 call memo → 同好交換情報(你 call ABC 我 call DEF)。沒有一步登天,靠累積。
但邊際效益會遞減,甚至受害於資訊太多變雜訊。某大分析師說他不看投片數字 — 那叫 noise(一下上修一下下修意義在哪)。2023-24 掌握封裝段數字能賺,現在這招鈍化了;抓大方向、知道哪些廠商吃大單就好,甚至看新聞就夠。資訊不是絕對必要、考驗眼光。

洗手場現在會認錯追回

洗手場一定有且很常見:達到條件停利/停損出去後立刻開走 = 洗手場。
以前被動接手,現在會把它追回來 — 追回的部位就洗不掉了,就算跌掉 20% 也巴在上面,甚至追在漲停價也沒關係。蠻敢認錯追回。
資訊靠累積但邊際效益遞減,抓大方向勝過追投片數字(noise)。被洗出場後若看好就認錯追回、追回就抱住(跌 20% 也不放)。

有餘裕的人很自在 + 別當炫耀貶低的暴發戶

商務艙:有餘裕很自在

第一次搭商務艙很新鮮想把飲料都點一次,現在上去直接睡覺、有些餐不吃;孩子短程還是搭經濟艙。
資產有餘裕的人很自在:可吃路邊攤也可吃貴的、可商務艙也可包機也可經濟艙,看當下感受。一味追求「沒什麼我不做、不吃、不喝」的,是才剛起來的暴發戶,自卑就會變這樣。

價值觀崩壞?別當討人厭的人

現在但凡投報率不高的生意都說不如放股票、萬物飛漲連價值觀也飛漲。主持人:現在還是賺錢時間點(搞不好持續到 2027/2028)先尻錢,空頭再來竹林七賢清談人情道德。
創投界最常被問「為什麼投你不投台積電」是幹話 —你要不就去投台積電,來做創投就是要 go big 賭大的。沒錢的最愛講「什麼我不投/不吃/不搭」顯得屌;真有錢的會分散、做創投是理念或好玩。
有餘裕的人很自在(路邊攤到包機都行);別當炫耀「什麼我不做」或貶低人的暴發戶(房地產好時也一堆無腦買房)。別 flashy(台灣相對安全所以很多人愛 po 甚至造假),專注自身價值跟過去比。
Q&A 心態 + 教養
9/10

隨時抽身不被影響 + 注意力是重要資產

停播感想:做好隨時抽身的準備

不是要塑造跟大家分開的氛圍;4-5 年如一日態度都一樣。維持這心態的好處是不太怕失去 —不是做好失去一切的準備,是做好隨時可以抽身、不被影響的準備。
節目/小投資哪天不做都知道找誰接手;用中性、過客的態度看它(它會來也可能會走),不會死抱著一定要維持它。一定會有說再見的一天,但不是立刻。

注意力是重要資產(玻璃心補充)

把注意力當成重要資產 — Meta/Google 都想買、很值錢,別浪費在不值得的人身上。YT/FB 看到白痴直接封鎖、別吵架(那是揮霍注意力);馬特鴻鋒不會管誰按一星(山太高、太冷、沒酒吧)。mute 掉負面的人(line 靜音、不該見的朋友家人也減少接觸)。

教養:孩子不必討好每個人

老婆罵醒他:「為什麼兒子要去討好不認識的人?那是你朋友不是他朋友。」孩子做到有禮貌(打招呼、正眼看人)就好,不必態度多好、不必抱抱。讓身邊每個人都好受,你就是最難受的那個 —能量是守恆的。
對節目/投資都用「隨時抽身不被影響」的中性態度。注意力是資產,mute 負面的人別浪費注意力。教養:孩子有禮貌就好,不必討好每個人。
