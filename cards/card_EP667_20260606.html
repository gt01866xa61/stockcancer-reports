老黃喊 Marvell 飛天,光是 AI 未來主軸

Computex 最大震撼:老黃講 Marvell 可能是下一個造園企業 — Marvell 像大牛車突然變飛天神牛直接射出去。

機制(老黃點石成金):主持人完全沒預期到。Marvell 在 DSP 不錯,但 ASIC 還在發展路上,沒 MediaTek/Broadcom 那麼明確。
→ 老黃喊盤能力極強,踢到誰誰就噴。本來主持人假設 Marvell 是「某程度反應」(看估值推到哪),意外被 Jensen 加持直接拉出去。股票有時有超出想像的發展。

光是未來主軸(非零和)

NVIDIA + Marvell 演講核心都強調光:RAC 內、跨 RAC、Data Center 間連線。
雖然 NVIDIA 說「能用銅就用銅」,但光的成長速率與可能性持續拉大,已形成強勁共識,資金可能再往光跑。重點:光跟銅非零和、一起成長(光成長性好一點),別用「一個起另一個死」的零和思維看市場。

AIPC + 地端模型
4/10

AIPC 第二次挑戰 — 地端模型可能省下巨量 token fee

NVIDIA 端出 DGX Spark(地緣產品 N1X)。第一批高通 AIPC 完全不成氣候、只是好玩談資;這次經一兩年算力+技術+軟體優化,來觀察地端模型能不能真的用。

地端模型機制(Snowflake 員工分享):
開源模型約落後閉鎖式半年 — AI 指數成長半年差很多,但爬到 sweet spot 後對絕大多數個人/企業就夠用,只有 cutting edge 研究、前端馬農需要最高階。
→ 若那天到來:買一台夠好的地端伺服器或 AIPC 跑地端模型,省大量 token fee。閉鎖模型拿 1 萬分,但人類需求 9000 分,開源追上就能滿足。
Snowflake 的 Data Lake 模式:用戶提問先傳 LLM 理解語境(這段外包、只抽一點),爬資料時用自己 Data Lake +內建模型(這段錢他賺)。付給 LLM 的費用若地端化會變成它的存異。

AIPC 概念與用料

概念股: NVIDIA NVDA MediaTek 2454 市場從賭資產股(辦公室設哪)變成「看老黃手上拿哪牌筆電就衝進去鎖漲停」
AIPC 用料較好(晶片端+板子被動元件各種元件材料提升),ASP 高、一台抵傳統消費 1.5 台。若銷量比之前增速,可 offset 今年手機筆電的弱勢。題材可幫 re-rate,但實務帶動筆電 OEM 要再觀察。
非零和:地端崛起不代表 Data Center 死,用量仍成長好幾倍,但很多使用會落到地端。觀察 AIPC 第二次挑戰能否成,以及用料提升的供應鏈。

NVIDIA 論述與時俱進,大龍頭可能擔綱最後一棒

NVIDIA 論述不停與時俱進:從「買越多省越多瘋狂建置」→ Agentic → Robotics → 生活應用;從賣 GPU → 賣 Horax Solution → 現在賣所有 Infra 的供應商。

主持人猜:會不會他們擔綱最後一棒?大家都知道 TSM/NVDA 好,但去買小標股(因為比較會動)。類比比特幣 — 多少人真信它未來?大家只是覺得它會跑很快。市場百花齊放時都想找跑最快的,所以先往別的地方跑。

盤勢:稱估值 vs 上修動能

這波盤石碰到這階段,「稱估值」的可能性遠大於找到新的上修動能 — 上修動能應該還沒有,要有技術突破才能把數字往上調。偏向「盤面好+資金多」持續往上推,不是基本面上修。市場討論已蠻充分反應。
大龍頭便宜但市場愛追會動的小標股 — 留意 NVDA/TSM 是否在投信資金進場後補漲。但要意識到這階段是「稱估值」推升,非新動能,需技術突破才會再上調數字。

再平衡時間點(看佔比還是週期性)

不會沒事把長得好好的東西剪掉 — 再平衡要有目的性。
通常因為:① 有東西要買(像 SpaceX 要上、要參加申購或二級市場買),沒錢就壓低部位替款 ② 真認為市場 FOMO,減碼去買剛掉下來的。其他時間點不一定會沒事調整,不是有事沒事就亂殺。
回檔連十日線沒破不出、成本遠凹月線;碰月線幾天沒站回才減碼。再平衡要有明確目的(籌錢買新標的/FOMO換倉),別沒事剪好部位。

品酒 human algorithm + 威航往事的化悲憤為力量

品酒像學語言 → human algorithm

自費去樂花品酒會,本來自卑怕丟臉只想躲角落,結果被熱情大哥大姐引導,終於聽懂品酒像學語言 — 不懂時覺得形容味道的是裝逼仔,懂了之後真能聞出金針菇(巔峰期下滑的酒)、茶味(乾掉後)、果香松露土壤。
悟出 human algorithm:把味覺嗅覺語言參透後,能在腦中跑演算法,立刻聯想推薦對方(自己/愛的人/新朋友)會喜歡的東西 — 像 Amazon 購物車「猜你喜歡」。
用節目輸出檢驗真懂:「我只要有辦法在節目跟大家分享、講得出來,代表我真的懂了;講不出來,很多時候是你看完以為懂了,但不一定能理解。」

被掐脖子無能為力(威航往事)

威航(父親航空)摔兩架飛機,要逼簽自願離職省資遣費,跑來訓練基地各種理由幹他們罵廢物。
當時死老百姓沒管道只能忍耐;有錢有餘裕後一定會修理這種人(年輕沒資源沒時間上法院,人家比你有餘裕)。路見不平拔刀相助 — 這種事發生在朋友家人小孩身上,就去修理那個人。那把火會變成督促往前的能量(化悲憤為力量)。
學習像學語言,參透後成識人演算法;最高指導原則是「能不接觸人就不接觸」(接觸人常讓自己失望)。年輕無資源只能忍,有餘裕才能伸張正義。
